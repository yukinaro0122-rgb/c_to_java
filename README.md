# c_to_Java

c資産をJava化するために使用するリポジトリです。

## ディレクトリ構成

```
.
├── meetings/             # 議事録等
├── assets/               # 画像ファイル等
├── docs/                 # 設計ドキュメント等の資料を配置
│   ├── projectA/         # pocプロジェクト単位の資産を配置
│   └── projectB/
├── src/                  # ソースコード等のプログラムを配置
│   ├── projectA/         # pocプロジェクト単位の資産を配置
│   └── projectB/
├── .github/              # GitHub 用設定 (Issue/PR テンプレートなど)
├── .gitignore            # pycache等commit非対象ファイルの定義ファイル
├── .env                  # 環境変数等の定義ファイル
└── README.md             # このファイル
```
